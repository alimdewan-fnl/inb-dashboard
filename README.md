# INB Dashboard

A responsive web dashboard for displaying key performance indicators (KPIs) and metrics for the INB project.

## 🔗 Live Demo
The dashboard is automatically published and hosted via GitHub Pages:
**[View Live Dashboard](https://alimdewan-fnl.github.io/inb-dashboard/)**

## 💻 How to Run Locally
Since this is a client-side web application built with HTML, CSS, and standard JavaScript, no special server setup is required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alimdewan-fnl/inb-dashboard.git
   ```
2. **Open the project foldler.**
3. **Double-click `index.html`** to open it directly in any modern web browser (Chrome, Safari, Firefox, Edge).

## 📂 Project Structure
- `index.html` - The main dashboard file containing all the layout, styling, and logic.

*Note: For the best experience, ensure JavaScript is enabled in your browser as it powers the interactive elements and charts.*
